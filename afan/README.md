# aafan (just for fun)

### Getting Start

A translate tools for Chinese-English used in command line.

It's easy and convenient to use.

It's light and small without dependence.

### Install

```
npm install -g afan

```

### Usage

```
$afan hello
你好
n. 表示问候， 惊奇或唤起注意时的用语
int. 喂；哈罗
n. (Hello)人名；(法)埃洛


```

```
$afan 你好
How are you
hello；hi

```
